# A1.3 German Course Summary

## Level Overview
A1.3 completes the A1 level, introducing: the perfect tense (Perfekt), two-way prepositions with location vs. direction, dative verbs (gefallen, passen, stehen), demonstrative/interrogative articles, and dative-only prepositions. Topics include housing, clothing shopping, education/career, and travel.

**Prerequisites:** A1.1-A1.2 grammar (present tense, modal verbs, separable verbs, possessives, basic dative)

---

## Grammar

### The Perfect Tense (Perfekt)
The main past tense in spoken German. Structure: **haben/sein (conjugated) + Partizip II (at end)**

- *Ich **habe** Deutsch **gelernt**.* (I learned German.)
- *Er **ist** nach Berlin **gefahren**.* (He went to Berlin.)

#### Partizip II Formation

**Regular verbs: ge- + stem + -t**
| Infinitive | Partizip II |
|------------|-------------|
| lernen | gelernt |
| machen | gemacht |
| kaufen | gekauft |
| arbeiten | gearbeitet (stem ends in -t/-d: add -et) |

**Irregular verbs: ge- + changed stem + -en**
| Infinitive | Partizip II |
|------------|-------------|
| schreiben | geschrieben |
| lesen | gelesen |
| sprechen | gesprochen |
| finden | gefunden |
| gehen | gegangen |
| fahren | gefahren |
| kommen | gekommen |
| bleiben | geblieben |
| sehen | gesehen |
| geben | gegeben |
| nehmen | genommen |
| essen | gegessen |
| trinken | getrunken |
| schlafen | geschlafen |
| helfen | geholfen |
| treffen | getroffen |
| sein | gewesen |
| haben | gehabt |
| werden | geworden |
| fliegen | geflogen |

**Verbs ending in -ieren: no ge-, just -t**
| Infinitive | Partizip II |
|------------|-------------|
| studieren | studiert |
| telefonieren | telefoniert |
| fotografieren | fotografiert |
| reservieren | reserviert |

**Separable verbs: prefix + ge + stem + -t/-en**
| Infinitive | Partizip II |
|------------|-------------|
| aufstehen | aufgestanden |
| einkaufen | eingekauft |
| anfangen | angefangen |
| mitkommen | mitgekommen |

| abfahren | abgefahren |

**Inseparable prefixes (be-, emp-, ent-, er-, ge-, ver-, zer-): no ge-**
| Infinitive | Partizip II |
|------------|-------------|
| besuchen | besucht |
| verstehen | verstanden |
| bekommen | bekommen |
| erzählen | erzählt |

#### haben or sein?

**Most verbs use haben:**
- *Ich **habe** gearbeitet.*
- *Er **hat** ein Buch gelesen.*
- *Wir **haben** gegessen.*

**Verbs of movement or change of state use sein:**
| Verb | Partizip II | Example |
|------|-------------|---------|
| gehen | gegangen | Ich **bin** gegangen. |
| fahren | gefahren | Sie **ist** gefahren. |
| fliegen | geflogen | Er **ist** geflogen. |
| kommen | gekommen | Wir **sind** gekommen. |
| laufen | gelaufen | Sie **ist** gelaufen. |
| reisen | gereist | Ich **bin** gereist. |
| aufstehen | aufgestanden | Ich **bin** aufgestanden. |
| ankommen | angekommen | Er **ist** angekommen. |
| einschlafen | eingeschlafen | Sie **ist** eingeschlafen. |
| werden | geworden | Er **ist** Arzt geworden. |
| bleiben | geblieben | Ich **bin** geblieben. |
| sein | gewesen | Er **ist** krank gewesen. |

**Note:** For sein/haben, Präteritum (war/hatte) is preferred over Perfekt in speech.

### Two-Way Prepositions (Wechselpräpositionen)
These 9 prepositions take **dative for location (Wo?)** or **accusative for direction (Wohin?)**:

| Preposition | English |
|-------------|---------|
| in | in, into |
| an | at, on, to |
| auf | on, onto |
| über | over, above |
| unter | under |
| vor | in front of |
| hinter | behind |
| neben | next to |
| zwischen | between |

#### Location (Wo?) = Dative
| Gender | Article | Example |
|--------|---------|---------|
| Masculine | dem | Das Buch liegt auf **dem** Tisch. |
| Feminine | der | Die Lampe steht in **der** Ecke. |
| Neuter | dem | Das Bild hängt an **dem** Fenster. |
| Plural | den (+n) | Die Stühle stehen an **den** Fenstern. |

#### Direction (Wohin?) = Accusative
| Gender | Article | Example |
|--------|---------|---------|
| Masculine | den | Ich stelle den Stuhl an **den** Tisch. |
| Feminine | die | Ich lege das Buch auf **die** Kommode. |
| Neuter | das | Ich hänge das Bild an **das** Fenster. |
| Plural | die | Ich stelle die Blumen auf **die** Fensterbank. |

#### Contractions
| Full Form | Contraction |
|-----------|-------------|
| in + dem | im |
| in + das | ins |
| an + dem | am |
| an + das | ans |

Examples:
- *Das Sofa steht **im** Wohnzimmer.* (location - dative)
- *Ich gehe **ins** Wohnzimmer.* (direction - accusative)

### Position Verbs

**Location verbs (Wo? + Dative) - where something IS:**
| Verb | Meaning | Example |
|------|---------|---------|
| liegen | to lie (flat) | Das Buch **liegt** auf dem Tisch. |
| stehen | to stand | Der Schrank **steht** an der Wand. |
| hängen | to hang | Das Bild **hängt** über dem Sofa. |
| sitzen | to sit | Die Katze **sitzt** auf dem Stuhl. |

**Action verbs (Wohin? + Accusative) - where something GOES:**
| Verb | Meaning | Example |
|------|---------|---------|
| legen | to lay, put flat | Ich **lege** das Buch auf den Tisch. |
| stellen | to put upright | Ich **stelle** die Vase auf das Regal. |
| hängen | to hang | Ich **hänge** das Bild an die Wand. |
| setzen | to set, put | Ich **setze** das Kind auf den Stuhl. |

### Dative-Only Prepositions
These prepositions **always** take dative:

| Preposition | Meaning | Example |
|-------------|---------|---------|
| aus | from, out of | Ich komme **aus der** Schweiz. |
| bei | at, with | Ich wohne **bei meinen** Eltern. |
| mit | with, by | Ich fahre **mit dem** Zug. |
| nach | to (cities/countries) | Ich fliege **nach** Spanien. |
| seit | since, for | Ich lerne **seit einem** Jahr Deutsch. |
| von | from, of | Ich komme **vom** Bahnhof. |
| zu | to | Ich gehe **zum** Arzt. |
| ab | from (time) | **Ab** morgen bin ich im Urlaub. |

#### Contractions
| Full | Contraction |
|------|-------------|
| bei + dem | beim |
| von + dem | vom |
| zu + dem | zum |
| zu + der | zur |

#### nach vs. zu vs. in (for "to")
| Preposition | Used for | Example |
|-------------|----------|---------|
| nach | cities, countries without article | Ich fahre **nach** Berlin/Deutschland. |
| in + acc. | countries with article | Ich fahre **in die** Schweiz/Türkei. |
| zu | people, buildings | Ich gehe **zum** Arzt / **zur** Bank. |
| an + acc. | bodies of water | Wir fahren **ans** Meer. |

### Time Expressions with Prepositions

**seit + dative (since, for - ongoing):**
- *Ich lerne **seit einem Jahr** Deutsch.* (I've been learning German for a year.)

**vor + dative (ago):**
- ***Vor zwei Wochen** war ich im Urlaub.* (Two weeks ago I was on vacation.)

**ab + dative (from - future):**
- ***Ab morgen** habe ich Urlaub.* (From tomorrow I'm on vacation.)

**von ... bis (from ... to):**
- *Ich habe **vom** 1. **bis zum** 15. Juli Urlaub.*

### Dative Verbs
Some verbs require the dative case for the person affected:

**gefallen (to please / to like):**
- *Die Jacke **gefällt mir**.* (I like the jacket.)
- ***Gefällt dir** das Hemd?* (Do you like the shirt?)

**passen (to fit):**
- *Die Hose **passt mir** gut.* (The pants fit me well.)

**stehen (to suit):**
- *Die Farbe **steht dir** gut.* (The color suits you.)

**helfen (to help):**
- *Kann ich **Ihnen** helfen?* (Can I help you?)

#### Word Order with Dative Verbs
Dative object often comes before nominative subject:
- ***Mir** gefällt die Jacke.* (I like the jacket.)
- *Gefällt **dir** das?* (Do you like that?)

### Demonstrative Articles: dieser/dieses/diese (this)
| Case | Masc. | Neut. | Fem. | Plural |
|------|-------|-------|------|--------|
| Nominative | dieser | dieses | diese | diese |
| Accusative | diesen | dieses | diese | diese |
| Dative | diesem | diesem | dieser | diesen |

Examples:
- ***Dieser** Pullover ist schön.* (This sweater is nice.)
- *Ich nehme **dieses** Kleid.* (I'll take this dress.)
- ***Diese** Schuhe gefallen mir.* (I like these shoes.)

### Interrogative Articles: welcher/welches/welche (which)
| Case | Masc. | Neut. | Fem. | Plural |
|------|-------|-------|------|--------|
| Nominative | welcher | welches | welche | welche |
| Accusative | welchen | welches | welche | welche |
| Dative | welchem | welchem | welcher | welchen |

Examples:
- ***Welcher** Rock gefällt dir?* (Which skirt do you like?)
- ***Welche** Größe brauchen Sie?* (Which size do you need?)

## Key Phrase Patterns

### Housing
- Wie groß ist die Wohnung?
- Die Wohnung hat ... Quadratmeter.
- Wie viel kostet die Miete?
- Das Sofa steht an der Wand.
- Ich stelle den Tisch ans Fenster.

### Past Events (Perfekt)
- Was hast du gestern gemacht?
- Ich habe ... gelernt/gearbeitet/gekauft.
- Ich bin nach ... gefahren/geflogen.
- Hast du schon...?
- Er ist Arzt geworden.

### Shopping for Clothes
- Kann ich Ihnen helfen?
- Ich suche eine/einen/ein...
- Welche Größe haben Sie?
- Kann ich das anprobieren?
- Die Jacke gefällt mir (nicht).
- Die Hose passt mir (nicht).
- Diese Farbe steht Ihnen gut.
- Das ist zu teuer/groß/klein.

### Education/Career
- Was hast du studiert?
- Ich habe ... studiert.
- Wo hast du gearbeitet?

### Travel

- Vor zwei Wochen war ich in Berlin.

## Sentence Complexity Guidelines for A1.3

**DO use:**
- Perfect tense (Perfekt) with haben and sein
- Partizip II (regular, irregular, separable, inseparable, -ieren)
- Two-way prepositions with correct case (location = dative, direction = accusative)
- Position verb pairs (liegen/legen, stehen/stellen, hängen/hängen, sitzen/setzen)
- Dative verbs (gefallen, passen, stehen, helfen)
- Demonstrative articles (dieser/dieses/diese)
- Interrogative articles (welcher/welches/welche)
- Dative-only prepositions (aus, bei, mit, nach, seit, von, zu, ab)
- Time expressions with seit, vor, ab
- Colors as adjectives

**DO NOT use:**
- Präteritum for verbs other than sein/haben
- Subordinate clauses (weil, dass, wenn, obwohl) - these are A2
- Relative clauses
- Comparative/superlative
- Passive voice
- Konjunktiv II (würde, könnte, etc.)

**Example appropriate sentences:**
- Ich habe gestern viel gearbeitet.
- Sie ist nach München gefahren.
- Das Buch liegt auf dem Tisch.
- Ich lege das Buch auf den Tisch.
- Diese Jacke gefällt mir sehr gut.
- Welche Größe brauchen Sie?
- Seit einem Jahr lerne ich Deutsch.
- Vor zwei Wochen war ich im Urlaub.
- Er ist Ingenieur geworden.
